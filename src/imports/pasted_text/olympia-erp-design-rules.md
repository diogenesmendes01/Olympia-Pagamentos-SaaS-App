OLYMPIA PAGAMENTOS — REGRA FINAL DE DESIGN DO ERP

Regra central da marca

A identidade da Olympia no ERP deve seguir esta lógica:

azul profundo = estrutura, ação, confiança
dourado envelhecido = valor, premium, destaque especial
marfim = apoio institucional
neutros claros = base do sistema
cores semânticas = só para feedback e status

Em resumo:
a marca dá personalidade
e as cores funcionais dão clareza

Paleta principal oficial

Cores da marca:

Azul principal: #1F3A5F
Dourado envelhecido: #C8A96B
Marfim: #F4EFE6

Neutros do produto:

Background geral: #F8FAFC
Superfície / cards: #FFFFFF
Borda padrão: #E2E8F0
Texto principal: #1E293B
Texto secundário: #64748B
Linha/divisor: #CBD5E1

Variações do azul:

Primary hover: #274872
Primary pressed: #162C47
Fundo azul suave: #EEF3F8
Regra de proporção visual

Para o ERP inteiro:

70% neutros claros
20% azul da marca
10% dourado + cores de status

Essa é a regra mais importante.

Se o sistema tiver muito azul escuro, fica pesado.
Se tiver muito dourado, fica datado.
Se tiver muita cor de status, vira bagunça.

Uso correto de cada cor

Azul profundo
Usar em:

sidebar
menu ativo
botões principais
links
ícones ativos
títulos importantes
estados de foco
gráficos principais

Dourado envelhecido
Usar apenas em:

selo premium
badges especiais
detalhes do logo
planos VIP
indicadores de valor
destaques estratégicos

Não usar dourado como:

cor principal de botão
fundo de tela
cor dominante de tabela
menu principal

Marfim
Usar em:

tela de login
landing interna
blocos institucionais
seções especiais
detalhes decorativos discretos

No dashboard principal, prefira fundo claro quase branco em vez de marfim em tudo.

Regra dos cards

Card padrão

fundo branco
borda leve
sombra muito discreta
raio médio

Não use card totalmente colorido como padrão.
O mais bonito para a Olympia é:

card branco
borda suave
detalhe colorido lateral ou no topo
título com cor do estado

Isso deixa o ERP mais premium.

Regra dos gráficos

No ERP, gráfico precisa ser bonito e legível.

Ordem de prioridade das cores:

azul principal para série principal
azul claro para comparação
dourado para valor estratégico
verde para resultado positivo
terracota para problema
âmbar para atenção

Não usar mais de 5 cores fortes no mesmo gráfico.
Se houver muitas categorias, usar neutros e destacar apenas o que importa.

Regra da tipografia

Logo:

Olympia = Cinzel SemiBold
Pagamentos = Montserrat Medium

Dentro do sistema:

usar Montserrat ou Inter
minha recomendação para ERP: Inter
Cinzel só no logo e em peças institucionais

Regra:
Cinzel não deve ser usada em:

menus
tabelas
botões
formulários
dashboards
Regra de estilo visual

O estilo geral do ERP deve ser:

limpo
elegante
sóbrio
corporativo
premium
tecnológico

Referência visual:
fintech premium com herança clássica sutil

Não fazer:

ornamentos gregos exagerados
colunas desenhadas dentro da interface
muito dourado
fundos pesados
degradês fortes em áreas funcionais

A Grécia antiga deve aparecer como sofisticação, não como fantasia temática.

Regra de cantos, sombras e espaçamento

Para o sistema ficar bonito:

raio dos cards: 12 a 16px
raio dos botões: 10 a 12px
raio dos inputs: 10 a 12px
sombra: leve e discreta
espaçamento interno generoso
bastante respiro entre blocos

Isso ajuda muito a deixar a marca premium.

Regra final de hierarquia

No ERP da Olympia, a hierarquia sempre deve ser:

neutros claros constroem a base
azul conduz a navegação e ações
dourado diferencia o que é premium
cores semânticas explicam status
tipografia limpa mantém legibilidade
Frase-regra para todo o time

Se você quiser uma regra única para dev, designer e produto, use esta:

Azul para ação e estrutura.
Dourado para valor e exclusividade.
Cinzas e branco para a base.
Outras cores apenas para feedback e status.

Resumo final pronto

Padrão oficial da Olympia para o ERP:

cor principal do sistema: azul profundo #1F3A5F
cor premium/destaque: dourado envelhecido #C8A96B
cor institucional de apoio: marfim #F4EFE6
base do produto: branco e cinzas claros
botão principal: azul
dourado só em destaque especial
alertas com cores semânticas suaves
cards e tabelas majoritariamente brancos
tipografia do sistema: Inter ou Montserrat
Cinzel apenas no logo
